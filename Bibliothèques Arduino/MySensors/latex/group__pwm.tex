\doxysubsection{Detailed Description}
Allows control of 2 independent P\+WM channels. A limited subset of G\+P\+IO pins can be connected to one of these 2 channels, allowing P\+WM control of G\+P\+IO pins. You have to set the desired pin into a particular Alt Fun to P\+WM output. See the P\+WM documentation on the Main Page.
