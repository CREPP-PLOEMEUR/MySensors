\doxysubsection{Detailed Description}
These options are specific to the n\+R\+F5 (with Enhanced Shock\+Burst) family of wireless transport modules. 

The n\+R\+F5 driver is O\+TA compatible with the R\+F24 driver.

The following chips are supported by this driver\+:
\begin{DoxyItemize}
\item n\+R\+F51x22
\item n\+R\+F52822 
\end{DoxyItemize}
